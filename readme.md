## CoSight Users

### Unzip dist folder

### Go to chrome extension page
```bash
chrome://extensions/
```

### Click 'Load unpacked' and choose the unzipped folder

## CoSight Developers

### Install backend dependencies
```bash
npm install
```

### Install client dependencies
```bash
cd frontend
npm install
```

### Add your own .env file
```bash
PORT = 5000
MONGODB_URL = ...
TOKEN_SECRET = ...
```

### Compile the extension
```bash
cd frontend
npm start
```
